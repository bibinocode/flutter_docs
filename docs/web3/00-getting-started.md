# 第 0 章：写给完全零基础的你 — 开始之前

> 如果你是一个完全的新手，不知道从哪里开始，这一章就是为你写的。我们不讲任何代码，只讲清楚 Web3 到底是什么、你需要准备什么、学习路线是怎样的。

## 0.1 Web3 到底是什么？用人话说

想象一下互联网的三个阶段：

```
Web1（1990-2005）：只读互联网
  你只能看网页，不能互动
  例子：新浪新闻、雅虎门户
  类比：图书馆 — 你只能看书，不能写书

Web2（2005-至今）：读写互联网
  你可以发微博、写评论、上传视频
  但是！你的数据属于平台（微信、抖音、微博）
  平台可以删你的帖子、封你的号
  类比：租房 — 你可以住，但房东说了算

Web3（正在发展中）：读写+拥有互联网
  你的数据属于你自己
  你的数字资产（代币、NFT）属于你自己
  没有人可以冻结你的账户
  类比：买房 — 房子是你的，没人能赶你走
```

### Web3 的核心理念

1. **去中心化**：没有一个公司控制一切
2. **所有权**：你真正拥有你的数字资产
3. **无需许可**：任何人都可以参与，不需要审批
4. **透明**：所有交易记录公开可查

### Web3 能做什么？（真实的应用场景）

| 领域 | Web2 的做法 | Web3 的做法 | 代表项目 |
|------|-----------|-----------|---------|
| 金融 | 银行转账，3天到账，手续费高 | 链上转账，秒到账，手续费低 | Uniswap, Aave |
| 艺术 | 在平台卖画，平台抽成 30% | 直接卖 NFT，创作者拿大头 | OpenSea, Blur |
| 游戏 | 游戏道具属于游戏公司 | 游戏道具是 NFT，属于玩家 | Axie Infinity |
| 社交 | 微博封号，内容消失 | 内容上链，永远存在 | Lens Protocol |
| 存储 | 文件存在阿里云，公司倒了就没了 | 文件分布式存储，永远在线 | IPFS, Filecoin |
| 身份 | 每个网站一个账号密码 | 一个钱包走天下 | ENS |

## 0.2 你需要准备什么？

### 硬件要求

- 一台电脑（Mac / Windows / Linux 都行）
- 8GB 以上内存（推荐 16GB）
- 稳定的网络连接

### 需要的前置知识

```
必须会的（没有这些基础很难学 Web3）：
├── 至少一门编程语言的基础（Go / JavaScript / Python 任选一个）
├── 基本的命令行操作（cd, ls, mkdir 这些）
├── Git 基础（clone, commit, push）
└── HTTP 基础（知道什么是 API、GET/POST 请求）

最好会的（会加速你的学习）：
├── JavaScript / TypeScript
├── React 或 Vue（前端框架）
├── Go 语言基础
├── Flutter / Dart
└── 数据库基础（SQL）

不需要提前会的（课程会教）：
├── Solidity（智能合约语言）
├── 区块链原理
├── 密码学
└── DeFi / NFT 概念
```

### 需要注册的账号

1. **GitHub** — 代码托管，必须有
2. **MetaMask** — 浏览器钱包插件，[metamask.io](https://metamask.io/)
3. **Alchemy** — 区块链节点服务（免费），[alchemy.com](https://www.alchemy.com/)
4. **Etherscan** — 区块浏览器 API（免费），[etherscan.io](https://etherscan.io/)

## 0.4 常见问题

### Q: 学 Web3 需要先学区块链底层原理吗？

A: 不需要精通，但需要理解基本概念。就像你不需要理解 TCP/IP 协议的每个细节才能做 Web 开发，但你需要知道 HTTP 请求是怎么回事。本课程第 1-3 章会带你快速过一遍必要的基础。

### Q: 我只会前端/后端，能学 Web3 吗？

A: 完全可以。Web3 开发本质上还是软件开发，只是多了一个"区块链"作为数据层。你已有的编程经验都能用上。

### Q: 需要买加密货币吗？

A: 学习阶段不需要花真钱。我们用测试网（Testnet），测试用的 ETH 可以从水龙头免费领取。

### Q: Web3 开发和传统开发最大的区别是什么？

A: 三个核心区别：
1. **数据存储在链上**：写入数据需要付 Gas 费，而且不可修改
2. **用户身份是钱包地址**：不需要用户名密码，用签名验证身份
3. **代码部署后不可修改**：智能合约一旦部署就无法直接修改（除非用代理模式）

### Q: 这个课程学完能找到工作吗？

A: Web3 开发者目前供不应求，尤其是会 Go + Solidity 的全栈开发者。完成本课程的三个实战项目后，你的 GitHub 就是最好的简历。
